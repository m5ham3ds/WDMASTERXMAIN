# WiFi Card Master Pro

Professional Android application for WiFi card testing and analysis.

## Features

- 🎯 Smart card generation with learning system
- ⚡ Fast background testing service
- 📊 Real-time statistics and analytics
- 💾 Local database storage (Room)
- 🔔 Advanced notification controls
- 🌙 Dark mode support
- 📤 Export results (CSV, JSON, TXT)
- 🧠 Pattern learning from successful tests

## Requirements

- Android 7.0 (API 24) or higher
- Android Studio Arctic Fox or newer
- Kotlin 1.9.20+
- Gradle 8.2+

## Setup

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle files
4. Run on device or emulator
